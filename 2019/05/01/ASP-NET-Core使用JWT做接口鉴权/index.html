发表于 2019-05-01 评论数：

关于jwt，不了解的同学可以先看看阮老师的介绍 jwt 介绍，讲得非常好。

为什么选择jwt token，传统的认证方式有什么问题？

jwt的最大好用是它是声明式，服务器无需保存任何session信息，而传统的session和cookie做认证，服务器都要保存状态信息，并不是stateless，而jwt可以是stateless。
